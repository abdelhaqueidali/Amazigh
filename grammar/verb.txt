<h2>Changes (Asnfel)</h2>
<div class="example">

<p>If the verb has one of the vowels <span class="tifinagh">ⴰ</span>, <span class="tifinagh">ⵉ</span>, <span class="tifinagh">ⵓ</span> at its beginning, the verbal marker for the pronoun <span class="latin-transliteration-word">netta</span> changes from <span class="tifinagh">ⵉ</span> to <span class="tifinagh">ⵢ</span>.</p>

  <p><span class="tifinagh">ⵓⵎⵥ</span> (<span class="latin-transliteration-word">"amẓ"</span> <span class="tifinagh">ⴳ ⵓⵙⵎⵉⴷ</span>) => <span class="latin-transliteration-word">netta </span><span class="tifinagh"><span style="color:red">ⵉ</span>ⵓⵎⵥ</span> => <span class="latin-transliteration-word">netta </span><span class="tifinagh"><span style="color:red">ⵢ</span>ⵓⵎⵥ</span>.</p>
  <p><span class="tifinagh">ⴰⵡⵉ</span> => <span class="tifinagh">ⴰⴷ <span style="color:red">ⵉ</span>ⴰⵡⵉ ⵏⵜⵜⴰ</span> => <span class="tifinagh">ⴰⴷ <span style="color:red">ⵢ</span>ⴰⵡⵉ</span>.</p>


<p>If the verb has <span class="tifinagh">ⵜⵜ</span> at its beginning, the verbal markers for the pronouns <span class="latin-transliteration-word">keyy</span>, <span class="latin-transliteration-word">kemm</span>, <span class="latin-transliteration-word">kenni</span>, <span class="latin-transliteration-word">kenninti</span> change from <span class="tifinagh">ⵜ</span> to <span class="tifinagh">ⵜⴻ</span>.</p>

  <p><span class="tifinagh">ⵜⵜⵓ</span> => <span class="latin-transliteration-word">keyy </span><span class="tifinagh"><span style="color:red">ⵜ</span>ⵜⵜⵓ<span style="color:red">ⴷ</span></span> => <span class="latin-transliteration-word">keyy </span><span class="tifinagh"><span style="color:red">ⵜⴻ</span>ⵜⵜⵓ<span style="color:red">ⴷ</span></span>.</p>
  <p><span class="tifinagh">ⵜⵜⵉⵏⵉ</span> => <span class="tifinagh">ⴰⴷ <span style="color:red">ⵜ</span>ⵜⵜⵉⵏⵉ<span style="color:red">ⴷ</span></span> => <span class="tifinagh">ⴰⴷ <span style="color:red">ⵜⴻ</span>ⵜⵜⵉⵏⵉⴷ</span>.</p>


<p>If the verb has <span class="tifinagh">ⵜⵜⴻⵜⵜ</span> at its beginning, the previous verbal markers <span class="tifinagh">ⵜ</span> for the pronouns <span class="latin-transliteration-word">keyy</span>, <span class="latin-transliteration-word">kemm</span>, <span class="latin-transliteration-word">kenni</span>, <span class="latin-transliteration-word">kenninti</span> are not added. This issue concerns <span class="tifinagh">ⵜⵜⵓ</span> in the imperfective form <span class="tifinagh">ⵜⵜⴻⵜⵜⵓ</span>.</p>

 <p><span class="tifinagh">ⵜⵜⴻⵜⵜⵓ</span> (<span class="latin-transliteration-word">"ttu"</span> <span class="tifinagh">ⴳ ⴰⵔⵓⵙⵎⵉⴷ</span>) => <span class="latin-transliteration-word">keyy </span><span class="tifinagh"><span style="color:red"></span>ⵜⵜⴻⵜⵜⵓ<span style="color:red">ⴷ</span></span>.</p>

<p>If the verb begins with <span class="tifinagh">ⵏⵏ</span>, we add <span class="tifinagh">ⵏⴻ</span> instead of <span class="tifinagh">ⵏ</span> only when conjugating the verb for the pronoun <span class="latin-transliteration-word">nekkni</span>.</p>
<p><span class="tifinagh">ⵏⵏⵉ</span>/<span class="tifinagh">ⵏⵏⴰ</span> (<span class="latin-transliteration-word">"ini"</span> <span class="tifinagh">ⴳ ⵓⵙⵎⵉⴷ</span>) => <span class="latin-transliteration-word">nekkni </span><span class="tifinagh"><span style="color:red">ⵏ</span>ⵏⵏⴰ</span> => <span class="latin-transliteration-word">nekkni </span><span class="tifinagh"><span style="color:red">ⵏⴻ</span>ⵏⵏⴰ</span></p>
</div>

<h2>Tenses</h2>
<div class="example">

<p><b>Past</b></p>
  <p>The perfective and negative perfective forms are used to talk about an action that happened in the past.  The name "perfective" indicates completion and finishing.</p>
  <p>Examples:</p>
  <p><span class="tifinagh"><span style="color:red">ⵖⵔⵉⵖ</span> ⴰⴷⵍⵉⵙ ⴰⴷ ⵉⴹⴳⴰⵎ</span>.</p>
  <p><b><span class="latin-transliteration-word">Ur</span></b> <span class="tifinagh"><span style="color:red">ⵖⵔⵉⵖ</span> ⴰⴷⵍⵉⵙ ⴰⴷ</span>.</p>
  <p>All other forms are negated by adding the particle <span class="latin-transliteration-word">ur</span> and using the same forms, except for the perfective, which has a negative perfective form to use with the negation particle <span class="latin-transliteration-word">ur</span>.</p>


<p><b>Present</b></p>
    <p>The bare form and the imperfective form are used with the present.</p>
      <p>Examples:</p>
      
      <p><span class="latin-transliteration-word">Righ ad </span><span class="tifinagh"><span style="color:red">ɣrɣ</span> adlis ad</span>.</p>
      
      <p><span class="latin-transliteration-word">Da </span><span class="tifinagh"><span style="color:red">qqarɣ</span> kra seg udlis ad kraygat ass</span>.</p>
    

<p><b>Future</b></p>
  
    <p>The bare and imperfective forms are used with the future.</p>
    <p><span class="latin-transliteration-word">Rad </span><span class="tifinagh"><span style="color:red">ɣrɣ</span> adlis ad</span>.</p>
    
    <p><span class="latin-transliteration-word">Rad </span><span class="tifinagh"><span style="color:red">qqarɣ</span> imikk seg udlis ad kraygat ass</span>.</p>
  

<p><b>Imperative</b></p>
  
    <p>Imperative forms are used to give a command. There is a regular imperative and an emphatic imperative.</p>
      
      <p><span class="tifinagh"><span style="color:red">Ɣr</span> adlis ad</span>!</p>
      
      <p><span class="tifinagh"><span style="color:red">Qqar</span> adlis ad</span>!</p>
    
<p><b>Contexts of the Subjunctive Form</b></p>
  
    <p>This form appears in different places, such as when using the particles <span class="latin-transliteration-word">lli</span>, <span class="latin-transliteration-word">da</span>, <span class="latin-transliteration-word">nna</span>.</p>
      <p>Perfective <span class="latin-transliteration-word">usmid</span>: <span class="tifinagh">afrux <b>lli</b> <span style="color:red">i</span>ɣra<span style="color:red">n</span></span></p>
      
      <p>Negative Perfective <span class="latin-transliteration-word">usmid anabaw</span>: <span class="tifinagh">afrux <b>da</b> ur <span style="color:red">i</span>ɣri<span style="color:red">n</span></span></p>
      
      <p>Imperfective <span class="latin-transliteration-word">arusmid</span>: <span class="tifinagh">afrux <b>nna</b> <span style="color:red">y</span>aqqra<span style="color:red">n</span></span></p>
    
</div>

<h2>Attention!</h2>
<div class="example">
    <p>When one of the verbal markers is the same letter as the one the verb begins with, a person often forgets to add the verbal marker without realizing that the letter in the verb is just an original letter in the verb and not a verbal marker:</p>
    <h3>Examples</h3>
      
      <p><span class="tifinagh">ⵉⵙⵙⴰⵏ</span> => <span class="tifinagh">ad issan<span style="color:red">n</span></span> (<span class="latin-transliteration-word">nitni</span>) ✅</p>
      <p> <span class="tifinagh">ad issan</span> (<span class="latin-transliteration-word">nitni</span>) ❌</p>
      
      
      <p><span class="tifinagh">ⵉⵍⵉ</span> => <span class="latin-transliteration-word">rad </span><span class="tifinagh"><span style="color:red">y</span>ili</span> (<span class="latin-transliteration-word">netta</span>). ✅</p>
      
      <p><span class="tifinagh">ⵉⵍⵉ</span> => <span class="tifinagh">ad ili netta</span> ❌</p>
      
      <p><span class="tifinagh">ⴼⴼⵖ</span> => <span class="latin-transliteration-word">nekk rad ffɣ<span style="color:red">ɣ</span></span> ✅</p>
      
      <p><span class="latin-transliteration-word">nekk rad ffɣ</span> ❌</p>
      
  
</div>
